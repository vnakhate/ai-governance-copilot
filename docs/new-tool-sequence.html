  1. Intent — employee asks to onboard a tool not in the registry.
  2. Collect — agent gathers only the missing fields (tool, data types, dept, jurisdiction).
  3. Confirm — agent summarizes; the user says "Yes".
  4. POST /request — classify data → risk tier → provisional assess() → missingAttributes → open_questions.
  5. Write + audit — a pending ai_vendor_request in Vanta, plus a requested audit event.
  6. Hand back — record id, provisional verdict, and the checklist of what review must resolve.
